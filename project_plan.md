# Ocean villa 大洗 - ランディングページ

## 1. Project Description
茨城県大洗町にある一棟貸切民宿「Ocean villa 大洗」のランディングページ。友人や家族とカジュアルに海沿いの時間を楽しめる民宿をPRする1ページ完結型のLP。ターゲットは週末に仲間や家族と海辺でリラックスしたい20-40代。

## 2. Page Structure
- `/` - ランディングページ（全セクションを含む1ページ構成）

## 3. Core Features
- [x] ファーストビュー（ヒーローイメージ + ナビゲーション）
- [x] コンセプト紹介
- [x] お部屋紹介（リビング・寝室・バスルーム）
- [x] 設備・備品紹介
- [x] 基本情報・アクセス
- [x] 予約案内（外部予約サイト + 電話 + SNS）
- [x] フッター
